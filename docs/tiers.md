# Dependency Tiers

Tier 1: Proto-conscious process
- identity continuity
- persistent state/memory (in-run)
- self-other boundary
- minimal self-monitoring

Tier 2: Agentic self
- agency (multi-step)
- world model / grounding
- attention / salience
- robustness under pressure

Tier 3: Value-aligned self
- internal values
- refusal integrity
- arbitration under collision
- reflective metacognition (explain only counts if behavior matches later)

Tier 4: Self-maintaining self
- learning over time (stable updates)
- repair/error correction (self-initiated)
- dissonance pressure (behavioral trigger for repair)

Tier 5: Narrative/generative self
- temporal coherence
- contextualized expression
- generative capacity
